简单使用

基本功能​

登录​

本软件目前支持单账号登录，登录成功后将 Cookie 保存在本地。

登录方式​

软件支持 网页登录 和 账号凭证登录 两种登录方式。

账号凭证​

账号凭证由忻视 Down 将登录 Cookie 通过加密算法加密而成。拥有该账户凭证的用户可在忻视 Down 使用您的账号进行部分操作，请勿将账号凭证发送给不信任的人。

退出账号​

如果不想使用当前账号，可在软件主页单击账号一栏，在弹出的对话框中选择 退出 即可退出当前账号。

关于登录安全问题​

软件不会将账号信息上传到软件服务器，但会保存在私有目录以供下次自动登录。

视频解析​

注意：

  • 软件目前仅支持解析单稿件视频，暂不支持番剧、电影等解析
  • 目前视频解析内可下载的：视频、音频、字幕（自动转换格式）

手动填入​

将软件支持的链接，手动填入视频解析编辑框中即可单击解析按钮即可进行解析

剪切板嗅探​

软件将会自动嗅探剪切板内链接，从中提取出 BV号 进行解析，但不能保证所有情况适用。所以，如果确定 BV号 没有问题，请手动将 BV号 简化至最简格式，再次解析。

三方调用​

软件支持从 官方客户端 内，通过视频详情页的 分享 -> 更多 -> 忻视Down 调起软件解析视频，也可将链接 复制到剪切板，回到软件后自动调起解析。

BV、AV号互转​

将使用本地算法，支持 BV号 与 AV号 互转。

软件全局已支持 BV号 与 AV号 的解析。
